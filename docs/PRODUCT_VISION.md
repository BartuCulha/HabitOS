# HabitOS — Product Vision

Saved from initial design session. See README.md for current build status.

## Decisions Made
- Claude only (primary) + Gemini (secondary) — both OAuth, no API keys
- Multi-user from day one (not exclusive to one person)
- Correlation data threshold: 3 days (not 30)
- Pantheon Council: included
- Mobile-first (React Native / Expo)
- App name: HabitOS
